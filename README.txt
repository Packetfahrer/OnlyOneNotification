A Simple tweak to disable multiple notifications turning on the Lockscreen. 
Supports iOS 7 as well as 64-bit systems.
Contains a few settings: enable/disable as well as sound and vibration.

Designed with THEOS, and should build out-of-the-box once you symlink theos into the project folder (ln -s /opt/theos theos on osx) I did not include my symlink for it because it may not work on your system.
